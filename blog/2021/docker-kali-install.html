Installing Docker CE on Kali Linux

18 July 2021

I have a similar post of installing Docker CE but on Red Hat 8. Installing on Kali Linux is not completely different but still documenting the steps I went through to set it up.

There's an existing docker package from apt therefore Kali-rolling repository named it docker.io

Install using apt

$ sudo apt install docker.io
Reading package lists... Done
Building dependency tree... Done
Reading state information... Done
The following additional packages will be installed:
  cgroupfs-mount containerd libintl-perl libintl-xs-perl libmodule-find-perl libmodule-scandeps-perl
  libproc-processtable-perl libsort-naturally-perl needrestart runc tini
Do you want to continue? [Y/n]

Post-installation

$ sudo groupadd docker
groupadd: group 'docker' already exists

$ sudo usermod -aG docker $USER

$ sudo systemctl enable docker
Synchronizing state of docker.service with SysV service script with /lib/systemd/systemd-sysv-install.
Executing: /lib/systemd/systemd-sysv-install enable docker
Created symlink /etc/systemd/system/multi-user.target.wants/docker.service → /lib/systemd/system/docker.service.

Starting and testing

Logout and then login back to update your permissions.

$ sudo systemctl start docker
