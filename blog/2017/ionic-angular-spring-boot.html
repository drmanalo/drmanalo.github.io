Spring Boot and Ionic

28 June 2017

While prototyping the best combination for Java and progressive web app, I have come across with ionic. The reason it caught my interest is it's Typescript support. I'm already considering phonegap but I think the first one has more to offer than the later. What I'm trying to accomplish is build once and publish it across multiple mobile platforms.

spring init

$ spring init -d=devtools,h2,data-jpa,data-rest,web progressive-web
Using service at https://start.spring.io
Project extracted to '/home/drmanalo/idea/progressive-web'

Install ionic and cordova

$ sudo npm install -g ionic cordova

After the installation, go to the progressive-web/src/main folder.
